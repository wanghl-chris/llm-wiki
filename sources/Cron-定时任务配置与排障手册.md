# Cron 定时任务配置与排障手册

## 一、概述

Hermes Agent 内置了 Cron 定时任务调度器，支持在指定时间自动执行任务，如：生成热点新闻图片、推送天气预报、生成知识简报等。

本手册记录了配置方法、已知问题和排障流程。

---

## 二、当前已配置任务

|----------|----------|----------|
| 每日 11:30 | 生成当日热点新闻图片 | 桌面「每日热点」文件夹 |
| 每日 17:00 | 生成猫狗宠物知识干货图片 | 桌面「每日热点」文件夹 |

## 三、核心特性

### 3.1 中国大陆节假日自动跳过
- 已集成 `chinese-holiday-workday-schedule` 技能
- 节假日当天自动跳过定时任务执行
- **重要**：每年初需要更新 `~/.hermes/scripts/is_holiday.py` 中的节假日集合

### 3.2 DNS 健康检查（解决 Mac 休眠问题）
- 任务执行前自动检查 DNS 连通性
- 依次测试：`open.feishu.cn` → `api.openai.com` → `dns.alidns.com`
- 每 15 秒重试一次，最多等待 5 分钟
- 网络未恢复则不执行任务，避免报错

### 3.3 飞书消息重试机制
- 第 1 次失败：等待 15 秒后重试
- 第 2 次失败：等待 45 秒后重试
- 第 3 次失败：等待 120 秒后重试
- 3 次重试全部失败才放弃

## 四、常见问题与排障

### 问题 1：Mac 休眠后任务不执行

**现象**：Mac 合上盖子休眠一段时间后，定时任务没有在预期时间执行。

**原因**：
- macOS 进入深度休眠后网络栈完全中断
- DNS 解析失败，导致任务执行失败
- 即使唤醒后，网络恢复也需要几十秒时间

**解决方案（已实施）**：
1. 在 `cron/jobs.py` 的 `tick()` 函数中增加 `_wait_for_network()` 检查
2. 任务执行前轮询 DNS，每 15 秒一次，最多等 5 分钟
3. 网络未恢复则跳过本次任务执行

**预防措施**：
- 重要任务尽量安排在 Mac 活跃时段执行
- 长时间离开前建议插上电源，避免深度休眠

---

### 问题 2：飞书消息发送失败

**现象**：任务执行了，但飞书没有收到消息。

**排查步骤**：
1. 检查飞书机器人 Token 是否有效
2. 检查网络是否能访问 `open.feishu.cn`
3. 查看 Cron 日志：`~/.hermes/cron/output/` 文件夹
4. 确认接收人是否在飞书应用的可用范围内

**常见原因**：
- 飞书应用权限变更
- 飞书机器人 IP 白名单未配置
- 网络代理变化

---

### 问题 3：节假日没有自动跳过

**现象**：节假日当天仍然执行了定时任务。

**原因**：
- `is_holiday.py` 中的节假日数据过期
- 当年的法定节假日安排未更新

**解决方案**：
1. 每年初根据国务院通知更新 `~/.hermes/scripts/is_holiday.py`
2. 更新 `HOLIDAYS` 集合（法定节假日日期）
3. 更新 `WORKDAYS` 集合（调休上班的周末日期）

---

### 问题 4：任务重复执行或不执行

**现象**：
- 同一个任务执行了多次
- 或所有任务都不执行

**排查**：
```bash
# 查看 Cron 进程
ps aux | grep cron

# 查看 Cron 日志
ls -la ~/.hermes/cron/output/

# 检查任务状态
hermes cron list
```

## 五、配置修改指南

### 修改任务执行时间

1. 打开交互式菜单
2. 选择「定时任务管理」
3. 选择「修改任务」
4. 输入新的 Cron 表达式（如 `0 8 * * *` 表示每天 8 点）

### 添加新任务

1. 打开交互式菜单
2. 选择「定时任务管理」
3. 选择「添加新任务」
4. 填写任务名称、执行时间、任务内容

### 禁用/启用任务

1. 打开交互式菜单
2. 选择「定时任务管理」
3. 选择「禁用任务」/「启用任务」

## 六、Cron 表达式速查

```
* * * * *
│ │ │ │ │
│ │ │ │ └── 星期 (0-7, 0=7=周日)
│ │ │ └──── 月份 (1-12)
│ │ └────── 日期 (1-31)
│ └──────── 小时 (0-23)
└────────── 分钟 (0-59)
```

**常用示例**：
- `0 8 * * *`：每天 8 点
- `30 11 * * 1-5`：工作日 11:30
- `0 9 * * 1,3,5`：周一、周三、周五 9 点
- `0 */2 * * *`：每 2 小时
